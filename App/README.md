# How to run:

## Using your phone with the expo app

1. clone
2. cd IB3-App
3. npm install
4. change the IP address to your pc's IP address (use ipconfig) in src/constants.js
5. npx expo start
6. scan qr-code with expo app

## Using an android emulator (Android Studio)

1. clone
2. cd IB3-App
3. npm install
4. npm run android
